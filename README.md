# Lullaby for the Going Merry

A small Next.js site presenting an original set of lyrics honoring the Straw Hat crew's first ship, the Going Merry.

## Getting Started

### Prerequisites

- Node.js 18 or later
- npm (bundled with Node.js)

### Setup

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to view the song.

## Scripts

- `npm run dev` – start the development server
- `npm run build` – create an optimized production build
- `npm start` – run the production build locally
- `npm run lint` – lint the project with Next.js defaults
